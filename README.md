# BrandSEO PrestaShop Module

BrandSEO creates SEO-ready brand/manufacturer landing pages connected to the PrestaShop catalog.

## Compatibility

- PrestaShop 1.7.x
- PHP 7.3+
- MySQL 5.7+

## Current status

MVP:
- Back Office tab.
- Manufacturer dashboard.
- Draft landing generation.
- Noindex by default.
- 301 disabled by default.
